Package Information

Package Name: angle
Package Type: Port
Version: chromium_5414
Description: A conformant OpenGL ES implementation for Windows, Mac and Linux.,The goal of ANGLE is to allow users of multiple operating systems to seamlessly run WebGL and other OpenGL ES content by translating OpenGL ES API calls to one of the hardware-supported APIs available for that platform. ANGLE currently provides translation from OpenGL ES 2.0 and 3.0 to desktop OpenGL, OpenGL ES, Direct3D 9, and Direct3D 11. Support for translation from OpenGL ES to Vulkan is underway, and future plans include compute shader support (ES 3.1) and MacOS support.

Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install angle

Manifest Mode:

 vcpkg add port angle
