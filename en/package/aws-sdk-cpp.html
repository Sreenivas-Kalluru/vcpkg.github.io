Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install aws-sdk-cpp

Manifest Mode:

 vcpkg add port aws-sdk-cpp


Supported Platforms and Architectures

Platforms: !(windows & arm) & !uwp
Architectures: , arm-uwp, arm64-windows, x64-linux, x64-osx, x64-uwp, x64-windows, x64-windows-static, x86-windows


Dependencies

  • aws-crt-cpp
  • curl(platform: !uwp & !windows)
  • openssl(platform: !uwp & !windows)
  • vcpkg-cmake (host: true)
  • vcpkg-cmake-config (host: true)
  • zlib
