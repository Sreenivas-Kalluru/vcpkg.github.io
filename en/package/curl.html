Features

• brotli : brotli support (brotli)
• c-ares : c-ares support
• http2 : HTTP2 support
• idn : Default IDN support
• idn2 : idn2 support (libidn2)
• mbedtls : SSL support (mbedTLS)
• non-http : Enables protocols beyond HTTP/HTTPS/HTTP2
• openssl : SSL support (OpenSSL)
• schannel : SSL support (Secure Channel)
• sectransp : SSL support (sectransp)
• ssh : SSH support via libssh2
• ssl : Default SSL backend
• sspi : SSPI support
• tool : Builds curl executable
• websockets : WebSocket support (experimental)
• winidn : WinIDN support
• winldap : LDAP support (WinLDAP). This feature does not include LDAPS support.
• winssl : Legacy name for schannel
• wolfssl : SSL support (wolfSSL)
• zstd : ZStandard support (zstd)


Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install curl

Manifest Mode:

 vcpkg add port curl
