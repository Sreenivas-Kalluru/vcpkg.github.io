Features

• lzma : Support LZMA compression in TIFF image files
• tools : Build tools
• webp : Support WEBP compression in TIFF image files
• zip : Support ZIP/deflate compression in TIFF image files
• zstd : Support ZSTD compression in TIFF image files


Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install tiff

Manifest Mode:

 vcpkg add port tiff
