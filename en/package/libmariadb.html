Features

• iconv : Enables character set conversion
• mariadbclient : Build the static mariadbclient library
• openssl : SSL support (OpenSSL)
• schannel : SSL support (Secure Channel)
• ssl : Default SSL backend
• zlib : Obsolete. zlib is always used. This feature is left for compatibility.
• zstd : Build zstd compression plugin.


Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install libmariadb

Manifest Mode:

 vcpkg add port libmariadb
