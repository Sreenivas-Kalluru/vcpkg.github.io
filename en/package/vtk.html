Features

• all : Build all vtk modules
• atlmfc : Mfc functionality for vtk on Windows
• cuda : Support CUDA compilation
• gdal : Support GDAL compilation
• geojson : Convert Geo JSON format to vtkPolyData
• mpi : MPI functionality for VTK
• opengl : All opengl related modules
• openvr : OpenVR functionality for VTK
• paraview : Build vtk modules required by paraview
• python : Python functionality for VTK
• qt : Qt functionality for VTK
• utf8 : Enables vtk reader/writer with utf-8 path support
• vtkm : Build with vtk-m accelerator and module.


Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install vtk

Manifest Mode:

 vcpkg add port vtk
