Features

• dbstat : Enable the DBSTAT virtual table
• fts3 : Enable the FTS3 extension
• fts4 : Enable the FTS4 extension
• fts5 : Enable the FTS5 extension
• geopoly : Enable geopoly functionality for sqlite3
• json1 : Enable JSON functionality for sqlite3
• limit : Enable the UPDATE/DELETE LIMIT clause
• math : Enable math functions
• memsys3 : Enable MEMSYS3
• memsys5 : Enable MEMSYS5
• omit-load-extension : Enable loading of external extensions
• rtree : Enable the RTREE extension
• session : Enable the SESSION extension
• tool : Build sqlite3 executable
• zlib : Build sqlite3 command line tool with zlib support; has no effect on the library itself


Installation Instructions

How to integrate vcpkg into your preferred environment:

  1. CMake / command line
  2. Visual Studio (CMake)
  3. Visual Studio (MSBuild)
  4. Visual Studio Code
  5. Manual integration (other build systems / IDEs)

Classic Mode:

 vcpkg install sqlite3

Manifest Mode:

 vcpkg add port sqlite3
